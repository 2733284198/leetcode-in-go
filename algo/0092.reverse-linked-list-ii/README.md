# [92. Reverse Linked List ii](https://leetcode-cn.com/problems/reverse-linked-list-ii/)

## 题目

反转从位置 m 到 n 的链表。请使用一趟扫描完成反转。

说明:
1 ≤ m ≤ n ≤ 链表长度。

示例1:

```
输入: 1->2->3->4->5->NULL, m = 2, n = 4
输出: 1->4->3->2->5->NULL
```


## 解题思路

1. 头插法反转链表, 每次将遍历的当前节点插入开始反转的位置。
2. 通过哨兵节点处理 m == 1 的情况
3. 通过m的值确定头插法的头节点位置 
4. 通过n-m的值确定执行几次头插操作



### 单链表反转常用的三种方法

1. 栈实现的链表反转
2. 递归实现单链表反转
3. 空间复杂度为 O(1) 单链表反转
    - 待翻转的结点。
    - 待反转结点的前驱结点 prev。
    - 待反转结点的后续结点 next。
