# [326. Power Of Three](https://leetcode-cn.com/problems/power-of-three/)

## 题目

给定一个整数，写一个函数来判断它是否是 3 的幂次方。

示例 1:

```
输入: 27
输出: true
```

示例 2:

```
输入: 0
输出: false
```

示例 3:

```
输入: 9
输出: true
```

示例 4:

```
输入: 45
输出: false
```

进阶：
你能不使用循环或者递归来完成本题吗？


## 解题思路


思路一：余数+商 循环迭代

时间复杂度：O(log (n))，在我们的例子中是 O(logn)。除数是用对数表示的。
空间复杂度：O(1)，没有使用额外的空间。


思路二：整数限制

```
n > 0 && 1162261467 % n == 0
```
