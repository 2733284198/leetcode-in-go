# [20. Valid Parentheses](https://leetcode-cn.com/problems/valid-parentheses/)

## 题目

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：

    左括号必须用相同类型的右括号闭合。
    左括号必须以正确的顺序闭合。

注意空字符串可被认为是有效字符串。


示例1:

```
输入: "()"
输出: true
```

示例2:

```
输入: "()[]{}"
输出: true
```

示例3:

```
输入: "(]"
输出: false
```

示例4:

```
输入: "([)]"
输出: false
```

示例5:

```
输入: "{[]}"
输出: true
```



## 解题思路

利用 栈 数据结构，匹配不上就入栈，匹配上就出栈，最后如果是空栈则说明为有效括号

## 总结
